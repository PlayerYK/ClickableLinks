Clickable Links Extension

Extension Overview

The Clickable Links Extension is a Chrome browser extension that automatically converts plain text URLs, email addresses, and various protocol links into clickable links on web pages.

How to Use

  1. Install the extension from the Chrome Web Store
  2. Click on the extension icon in the browser toolbar
  3. The extension will automatically process the current page, converting text links into clickable links

Test Case 1: HTTP and HTTPS URLs

Test Case 2: FTP and File URLs

Send an email: mailto:contact@example.com

Test Case 5: Magnet Links

Download torrent: magnet:?xt=urn:btih:123456789abcdef

Test Case 6: Mixed Content

Visit https://www.example.com, email info@example.com, or download from ftp://files.example.com. For local files, use file:///C:/path/to/file.txt. Magnet link: magnet:?xt=urn:btih:987654321fedcba

Privacy Statement

The Clickable Links Extension respects user privacy and does not collect, store, or transmit any personal information or browsing data. The extension only processes the text content of the current page when the user actively clicks the extension icon and does not run automatically or process data in the background.
